Q5-MoreDeclare

題目敘述

「變數」是程式語言非常重要的概念。一個變數包含四個部分：

  1. 名稱，例如：j
  2. 型態，例如：int
  3. 地址：這個變數在電腦記憶體中的位置
  4. 儲存值：例如：35

因此變數可以透過「=」這個運算子將數值「存入」變數中，也可以透過引用變數的名稱將變數的值「取出」。
請宣告:

  1. 型態是 int 的變數 i，將 3 存入 i 中；
  2. 型態是 char 的變數 c，將 ‘a’ 存入 c 中（注意用單引號）；
  3. 型態是 boolean 的變數 bool，將 true 存入 bool 中；
  4. 型態是 double 的變數 pi，將 3.14159 存入 pi 中；
  5. 型態是 String 的變數 s，將 “Hello” 存入 s 中（注意用雙引號）。

接著將這些變數的值輸出。

要求輸出

這是 format string 的使用範例： 3 a true 3.141590 Hello

public class MoreDeclare {
	public static void main(String[] args) {
		System.out.printf("這是 format string 的使用範例： %d %c %b %f %s\n", 3, 'a',true, 3.14159, "Hello");
		// format string 中以 % 起始的字是用來指定相對應變數
		// %d 輸出 int
		// %c 輸出 char
		// %b 輸出 bool
		// %f 輸出 double
		// %s 輸出 string
		// 內儲存值的輸出格式 \n 是換行符號
	}
}

題目敘述

Java 的 Math 套件（package）有許多數學上常用的函數。

請使用 Math.pow 及 Math.sqrt 寫一程式，讓使用者輸入兩點座標 (x1, y1) 和 (x2, y2)，分別為 (3, 4) 和 (12, 1)。

計算及顯示兩點之間的距離，距離值請取到小數第二位。

要求輸出

輸入第 1 組的 x 和 y 座標：
3 4
輸入第 2 組的 x 和 y 座標：
12 13

import java.util.Scanner;
public class Distance {
    public static void main(String args[]) {
        double x1, y1, x2, y2, distance;
        Scanner keyboard = new Scanner(System.in);  // 請參考上一題

        System.out.println("輸入第1組的x和y座標：");
        x1 = keyboard.nextDouble();
        y1 = keyboard.nextDouble();

Q11-CalCircle

題目敘述

請使用一個命名為 calCircle 的類別方法（static method）撰寫一輸出半徑是 5 的圓面積的程式。 calCircle 沒有輸入參數，也沒有傳回（return）結果至 main 中。

public class CalCircle {
    // 不變的常數使用 static final 宣告
    static final double PI = 3.14159;

        calCircle();  // 這是最簡單的方法呼叫：沒有傳入值，也沒有傳回值

        System.out.printf("半徑:%.2f 圓面積:%.2f",radius,radius*radius*PI);
    }
}

  • 傳入兩個整數(int, int)
  • 印出與傳回兩個整數的和(int)

宣告另一個 add 方法：

  • 傳入兩個浮點數(double, double)
  • 印出與傳回兩個浮點數的和(double)

宣告另一個 add 方法：

  • 傳入兩個字串(String, String)
  • 印出與傳回合併後的兩個字串(String)

  • add(2, 3);
  • add(5.2, 4.3);

Adding two doubles: 9.5
Adding two strings: I love Java!!
5 9.500000 I love Java!!

    }

        double totalarea;

        totalarea = calCircle(5) + calTriangle(10,5) + calSquare(5,10);
        System.out.printf("此圖形總面積為：%f\n",totalarea);
    }
    static double calCircle(int r) {
        return r*r*3.1415926;
    }
    static double calTriangle(int x, int y) {
        return x*y/2;
    }
    static double calSquare(int a , int b) {
        return a*b;
    }
}
